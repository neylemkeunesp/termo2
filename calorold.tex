\frame{ \frametitle{I Lei} Um corpo permanece em movimento
  retílineo uniforme a menos que atuem forças sobre ele.

\vspace{1 cm}
  Interpretação:
  \begin{itemize}
  \item Definição de referenciais inerciais.
  \item Definição do tempo para tornar o movimento simples.
  \item Dificuldade de observação direta.
  \item Não é uma lei do universo é uma definição.
  \end{itemize}
}

\frame{ \frametitle{II Lei} A Força que atua em um corpo é igual
  a variação do momento linear.

$$\vec{F}=\frac{d\vec{p}}{dt}$$

onde $\vec{p}=m\vec{v}$.  }

\frame{ \frametitle{Comentários}
  \begin{itemize}
  \item De acordo com Marion a segunda Lei de Newton é a
    definição de Força, não sendo portanto uma lei da
    natureza, mas sim uma definição.
  \item A definição é válida para referenciais inerciais.
  \item Observe que nos casos em que $m=cte$ recaímos na
    tradicional $\vec{F}= m\vec{a}$.
  \item De acordo com outras interpretações, Força é
    conceito antropomorfico, e a segunda lei é de fato um enunciado
    sobre a natureza.

\end{itemize}
}

\frame{ \frametitle{III Lei}

  Se dois corpos exercem forças um sobre o outro, as forças
  são iguais em módulo e direção mas opostas em sentido.

  Comentários:

  \begin{itemize}
  \item ``Para toda força de ação existe uma força de
    reação'', imagem popular nem sempre corresponde ao enunciado
    físico.
  \item Momento total é conservado. III Lei é uma lei de
    conservação.
  \item De acordo com Marion, a III Lei é a única que possui
    conteúdo físico.
  \item Definição de massa.
  \end{itemize}
}

\frame{ \frametitle{Massa Inercial e Gravitacional}
  \begin{itemize}
  \item Massa Inercial originária da III Lei.
  \item Massa Gravitacional originária da Lei da Gravitação
    Universal.
  \item Princípio do Correspondência: Massa Gravitacional=Massa
    Inercial.
  \end{itemize}
}

\frame{\frametitle{Trabalho e energia}


  Trabalho:

$$W=-\int_a^b \vec{F(x)}.d\vec{x}$$


{\bf Def.} Forças Conservativas:

Se a força é conservativa o trabalho não depende do caminho.
}


\frame{\frametitle{Energia Mecânica}

  {\bf Def.} Energia Cinética:

$$K=\frac{1}{2}m v^2$$

{\bf Def.} Energia Potencial: Se $F$ é conservativa então:

$$\int_\infty^x \vec{F(x^\prime)}.d\vec{x^\prime}$$


{\bf Def. Energia Mecânica}

$$E=U+V$$

}

\frame{\frametitle{História do Calor} O calor era pensado como uma
  quantidade conservada.

  Problemas:

  \begin{itemize}
  \item Mesma quantidade de calor aumentava a temperatura de forma
    diferente. Calor Específico.
  \item O Calor não podia ser inteiramente reaproveitado.
    Experimento de Ericsson em 1853.
  \item Calor por irradiação.
  \item Trabalho pode ser convertido em Calor.
  \end{itemize}

}


\frame{\frametitle{1^{a} Lei da Termodinâmica} A energia interna de
  um sistema:

$$U=q+w$$

onde $q$ é o calor trocado e $w$ o trabalho realizado é
conservada.

(Meyer 1842)

}


\frame{\frametitle{Teoria Cinética dos Gases}


  Os átomos possuem graus de liberdade que podem ser utilizados
  para armazenar energia. A teoria cinética consegue prever o
  comportamento de gases precisamente além de permitir interpretar
  a temperatura:

$$\frac{3}{2}kT=\frac{m\langle v^2 \rangle}{2}$$

$k$ constante de Boltzmann.  }


\frame{\frametitle{Teoria Quântica}

  A Teoria Quântica prvê que os níveis de energia que um
  sistema fechado pode possuir são discretos. Esta
  constatação tem importantes consequências para a
  Termodinâmica.

$$U=\sum_i N_i \epsilon_i$$
}


\frame{\frametitle{Por que o Calor Flui?}


  Em primeiro lugar quanto maior for a energia de um sistema, maior o
  número de estados acessíveis.

  {\bf Def. } Considere um sistema com $N$ partículas e com 3
  níveis possíveis de energia: $\epsilon=\epsilon_o,
  2\epsilon_o, 3\epsilon_o$.Suponha agora que temos 3 partículas e
  um valor de energia total de $3 \epsilon$. Quanto vale a
  multiplicidade $W$ deste sistema. E para o caso $W=6\epsilon_o$?



  \frame{\frametitle{Fluxo de Energia}

    A energia se distribui igualmente pois isto maximiza a
    multiplicidade. Considere dois sub-sistemas formado cada um por 3
    partículas e uma energia interna de $12\epsilon_o$. Calcule as
    multiplicidades para o caso em que a energia se distribui de forma
    simetrica e para o caso em que isto não ocorre. Se convença
    que a multiplicidade do primeiro caso é maior.  }



  \frame{\frametitle{Observações}
    \begin{itemize}
    \item O fluxo de energia maximiza a multiplicidade, mas nem sempre
      equaliza as energias.
    \item A tempertura é que acaba por ser igualada em quaisquer
      sistemas em contato. Sistemas em contato termico tendem a
      possuir a mesma temperatura,
    \end{itemize}
  }
