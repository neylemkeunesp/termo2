\section{Leis de Newton}

\frame{ \frametitle{I Lei} Um corpo permanece em movimento
  retílineo uniforme a menos que atuem forças sobre ele.

\vspace{1 cm}
  Interpretação:
  \begin{itemize}
  \item Definição de referenciais inerciais.
  \item Definição do tempo para tornar o movimento simples.
  \item Dificuldade de observação direta.
  \item Não é uma lei do universo é uma definição.
  \end{itemize}
}

\frame{ \frametitle{II Lei} A Força que atua em um corpo é igual
  a variação do momento linear.

$$\vec{F}=\frac{d\vec{p}}{dt}$$

onde $\vec{p}=m\vec{v}$.  }

\frame{ \frametitle{Comentários}
  \begin{itemize}
  \item De acordo com Marion a segunda Lei de Newton é a
    definição de Força, não sendo portanto uma lei da
    natureza, mas sim uma definição.
  \item A definição é válida para referenciais inerciais.
  \item Observe que nos casos em que $m=cte$ recaímos na
    tradicional $\vec{F}= m\vec{a}$.
  \item De acordo com outras interpretações, Força é
    conceito antropomorfico, e a segunda lei é de fato um enunciado
    sobre a natureza.

\end{itemize}
}

\frame{ \frametitle{III Lei}

  Se dois corpos exercem forças um sobre o outro, as forças
  são iguais em módulo e direção mas opostas em sentido.

  Comentários:

  \begin{itemize}
  \item ``Para toda força de ação existe uma força de
    reação'', imagem popular nem sempre corresponde ao enunciado
    físico.
  \item Momento total é conservado. III Lei é uma lei de
    conservação.
  \item De acordo com Marion, a III Lei é a única que possui
    conteúdo físico.
  \item Definição de massa.
  \end{itemize}
}

\frame{ \frametitle{Massa Inercial e Gravitacional}
  \begin{itemize}
  \item Massa Inercial originária da III Lei.
  \item Massa Gravitacional originária da Lei da Gravitação
    Universal.
  \item Princípio do Correspondência: Massa Gravitacional=Massa
    Inercial.
  \end{itemize}
}


\frame{ \frametitle{ReferenciaL Inercial}

  {\bf Def.:} Se existe um Referencial Inercial, um corpo sob o qual
  não atuem forças deve possuir velocidade constante em
  relação a este referencial.  }

\frame{
Exemplos:

\begin{itemize}
\item Estrelas Distantes.
\item Sob certas condições a Terra (velocidades baixas e
  distâncias curtas).
\end{itemize}

}

\frame{

  \frametitle{Técnicas de Resolução de Problemas}
  \begin{itemize}
  \item Faça um esquema do problema.
  \item Escreva as quantidades fornecidas.
  \item Escreva as equações úteis.
  \item Manipule as equações.
  \item Aplique os valores dados nas equações obtidas para
    determinar as equações buscadas.
  \end{itemize}
}

\frame{ 
\frametitle{Técnicas para testar os resultados}

  \begin{itemize}
  \item Não acredite em ninguém, especialmente em você mesmo.
    Chegar nas respostas é mais fácil do que ter certeza que
    elas estão corretas.
  \item Teste as unidades.
  \item Teste os limites do problema. O que ocorre quando
    $t\to\infty$, $m\to 0$ ou outros casos em que conhecemos a
    resposta.
  \item Compare seus resultados com simulações em computador. 
  \end{itemize}
}

\section{Teoremas de Conservação}



\frame{
\frametitle{Conservação do Momento Linear}
O momento linear de uma partícula é conservado se a força total
for nula. 

}


\frame{
\frametitle{Conservação do Momento Angular}

{\bf Momento Angular:} 
$$\vec{L}=\vec{r}\times \vec{p}$$

{\bf Torque:}
$$\vec{N}=\vec{r}\times \vec{F}$$

O momento angular  de uma partícula é conservado se o  torque total
for nulo. 


}

\frame{
\frametitle{Trabalho e Energia Potencial}
{\bf Trabalho:}
$$W_{1\to 2}=W_{12}=\int_1^2\vec{F}\cdot d\vec{r}$$


{\bf Energia Potencial - Campo Conservativo}
$$\int_1^2\vec{F}\cdot d\vec{r}=U_1-U_2$$

}
\frame{
\frametitle{Teorema Trabalho-Energia Cinética}
$$W_{12}=T_2-T_1$$
}

\frame{
\frametitle{Campo Conservativo}
\begin{center}
\includegraphics[scale=0.3]{campoconservativo}
\end{center}

{\bf Força Conservativa}
$$\vec{F}=-\nabla U$$

{\bf Teorema:}

Uma força é conservativa se e somente se $\nabla\times \vec{F}=0$. 
}


\frame{
\frametitle{Conservação da Energia Mecânica}

{\bf Energia Mecânica:}
$$E=T+U$$

A energia total de uma partícula em um campo conservativo é constante.

}
